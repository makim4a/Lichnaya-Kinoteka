## Описание

`Movie Library` — это простое десктопное GUI-приложение для ведения личной коллекции фильмов. Оно позволяет добавлять, просматривать и фильтровать фильмы, а вся информация хранится локально в файле `movielibrary.json`.

### Основные возможности:

*   **Добавление фильмов:** Ввод названия, жанра, года выпуска и рейтинга.
*   **Отображение:** Все фильмы показываются в удобной таблице.
*   **Валидация ввода:** Проверяется корректность вводимых данных (год — число, рейтинг — от 0 до 10).
*   **Фильтрация:** Возможность отфильтровать список по жанру или году выпуска.
*   **Сохранение данных:** Вся библиотека автоматически сохраняется в `movielibrary.json` при добавлении нового фильма и загружается при запуске.

## Технологии

*   **Язык:** Python 3
*   **GUI:** `tkinter` (стандартная библиотека Python)
*   **Формат данных:** JSON
*   **Система контроля версий:** Git

## Примеры использования (Как запустить)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL вашего репозитория>
    cd <папка репозитория>
    ```

2.  **Запустите приложение:**
    Убедитесь, что у вас установлен Python 3.
    ```bash
    python movie_library.py
    ```

3.  **Использование интерфейса:**
    *   **Добавление:** Заполните поля «Название», «Жанр», «Год выпуска» и «Рейтинг», затем нажмите кнопку «Добавить фильм».
    *   **Фильтрация:** Введите жанр или год в соответствующие поля в секции фильтрации и нажмите «Применить фильтр».
    *   **Сброс фильтра:** Нажмите кнопку «Сбросить», чтобы снова отобразить всю коллекцию.
    *   **Сохранение:** Данные сохраняются автоматически при добавлении фильма.
